Саня, надо скорректировать АПИ, мне нужно тебе передавать еще один параметр - название файла записи звонка. Также в интерфейсе заявки нужно будет добавить кнопку или ссылку на адрес вида http://<сервер телефонии>/api.php?f=get_record&name=<имя файла из бд, которое я тебе буду передавать>
для троицка запрос для получения записи:
http://voip.nppx.ru:8001/api.php?f=get_record&name=<имя файла>
запрос для исходящего вызова:
http://voip.nppx.ru:8001/api.php?f=call_out&from=901&to=<куда звонить.
в качестве from 901 - телефон диспетчера, вообще вместо хардкода нужно подставлять динамически внутренний телефон сотрудника, который кликает на ссылку. если диспетчеров 2 и у каждого свой телефон, то должно подставляться или 901 или 902 например.

webAPI для запросов к телефонии

API:
api.php?f=<method>&<param0>=...

Методы:
	get_record - получить запись файла
		параметры:
		name - имя файла записи (должно быть известно)
		
	call_out - совершить звонок (originate)
		параметры:
		from - кто звонит (Caller)
		to	 - куда звонить (Callee)
